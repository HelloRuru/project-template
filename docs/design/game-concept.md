# 遊戲概念設計

> 最後更新：2026-02-23

---

## 核心概念

玩家扮演一名即將入宮甄選的女子，從甄選那一刻起就開始宮鬥。透過智謀、人際關係、才藝培養，一步步在後宮中生存與晉升。

核心特色是**世代傳承**——培養子嗣，讓家族跨越世代延續權力與榮耀。

## 玩家角色

- **固定女性**（嬪妃視角）
- 不可選擇男性角色
- 從甄選入宮前開始遊戲

---

## 世代傳承機制

### 傳承流程

1. **第一代**：玩家從甄選入宮，目標是生存、晉升、生下子嗣
2. **第二代**：以女兒或媳婦的視角繼續遊戲，繼承母親的人脈與資源
3. **第三代以後**：家族累積的聲望、財富、秘辛成為新的遊戲資源

### 跨世代要素

| 要素 | 說明 |
|------|------|
| 家族聲望值 | 跨代累積，影響初始位份與人脈 |
| 傳承技能 | 琴藝、毒術、醫術等代代相傳 |
| 家族秘辛 | 可作為把柄或護身符 |
| 宮廷人脈網路 | 上一代建立的關係影響下一代 |
| 歷代恩怨 | 影響新一代的起始處境 |

### 傳承觸發條件

- 玩家角色年老退位
- 皇帝駕崩，朝代更替
- 玩家選擇主動傳承
- 玩家角色意外死亡（強制傳承）

## 位份晉升系統

| 位份 | 說明 |
|------|------|
| 答應 | 最低位份，剛入宮 |
| 常在 | 略有寵幸 |
| 貴人 | 開始有自己的宮女 |
| 嬪 | 可參與後宮大事 |
| 妃 | 掌管一宮事務 |
| 貴妃 | 後宮高位，權傾一方 |
| 皇貴妃 | 僅次於皇后 |
| 皇后 | 後宮之主 |
| 太后 | 終極目標，垂簾聽政 |

## 屬性系統

| 屬性 | 影響 |
|------|------|
| 容貌 | 皇帝寵幸機率、甄選表現 |
| 才藝 | 琴棋書畫，提升文化聲望 |
| 心計 | 識破陰謀、策劃行動成功率 |
| 體質 | 懷孕機率、抗毒能力、壽命 |
| 人緣 | 宮女忠誠、太監合作、妃嬪結盟 |
| 福運 | 隨機事件的正面傾向 |

## 行動系統（每日 / 每旬）

### 日常行動

- 練琴、讀書、刺繡、散步、禮佛

### 社交行動

- 拜訪其他嬪妃、結盟、挑撥離間

### 陰謀行動

- 下毒、造謠、賄賂太監、安插眼線

### 皇帝相關

- 爭寵、侍寢、同遊、進言

### 家族行動

- 接濟娘家、聯姻佈局、召見家人

### 子女行動

- 教育子女、安排聯姻、培養繼承人

## 人際關係系統

| 關係對象 | 影響 |
|---------|------|
| 皇帝 | 寵幸頻率、位份晉升 |
| 太后 | 後宮地位、保護傘效果 |
| 其他嬪妃 | 結盟 / 中立 / 敵對 |
| 宮女 | 情報蒐集、陰謀執行 |
| 太監 | 消息通報、辦事效率 |
| 娘家 | 外朝支援、家族勢力 |

---

## 子嗣系統

- **懷孕**：受體質、皇帝寵幸頻率、後宮安全影響
- **保胎**：防範其他嬪妃的暗算
- **生產**：男女隨機，影響後續策略
- **教育培養**：決定子女屬性成長方向
- **聯姻**：與朝臣家族聯姻，擴展勢力
- **傳承觸發**：玩家角色年老 / 駕崩 → 切換到下一代

---

## 甄選入宮（遊戲開局）

### 角色創建

- 家世背景選擇（影響初始屬性與人脈）
- 容貌、才藝、個性設定
- 初始技能分配

### 甄選過程

- 多輪篩選，每輪都可能被淘汰
- 表現影響初始位份
- 結識其他秀女（未來的盟友或敵人）

### 入宮分配

- 被分配到哪個宮殿
- 影響初期人際關係
- 決定第一個接觸到的高位嬪妃
